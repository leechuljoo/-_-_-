이차전지를 이해하기 위한 기본 구조와 원리
전지의 기본 구조
⊕

양극 (Cathode)

방전 시 환원 반응이 일어나는 전극. 높은 전기화학 전위를 가짐.
현재 주로 리튬전이금속산화물(LCO, NCM 등) 사용.

환원전극 LiCoO₂ 기반
⊖

음극 (Anode)

방전 시 산화 반응이 일어나는 전극. 낮은 전기화학 전위를 가짐.
상용 LIB에서는 흑연(Graphite) 사용.

산화전극 흑연(C₆)
💧

전해질 (Electrolyte)

이온의 이동 통로 역할. 양극과 음극 사이에서 리튬 이온(Li⁺)을 전달함. 전자는 통과하지 못함.

이온 전도체
🔲

분리막 (Separator)

양극과 음극의 물리적 접촉을 차단하면서도 이온은 통과시키는 다공성 막. 전해질을 함침하여 사용.

다공성 고분자막
전압 · 용량 · 에너지
⚡

전압 (Voltage, V)

양극 전위 − 음극 전위의 차이. 두 전극의 전기화학 전위 차이로 결정됨.

V = E(양극) − E(음극)
🔋

용량 (Capacity, Q)

저장된 전하의 양. 단위는 C 또는 Ah.
1 Ah = 1A의 전류로 1시간 방전 가능한 용량 (3600 C)

1 Ah = 3600 C
💡

에너지 (Energy, E)

일을 할 수 있는 능력. 단위는 J 또는 Wh.
18650 셀 (3.0Ah / 3.6V) = 10.8 Wh

E = V × Q (Wh)
📐

에너지 밀도

부피당(Wh/L) 또는 질량당(Wh/kg) 에너지의 양. LIB 현재 목표: Egrav > 400 Wh/kg

18650: ~260 Wh/kg, ~730 Wh/L
인터칼레이션 (Intercalation)

리튬이온전지 구동 원리 — 흔들의자(Rocking Chair)형

방전 · 음극 (산화)
LiₓC(s) → C(s) + xLi⁺(aq) + xe⁻
방전 · 양극 (환원)
Li₁₋ₓCoO₂ + xLi⁺ + xe⁻ → LiCoO₂(s)

Li⁺은 전해질을 통해, 전자(e⁻)는 외부 도선을 통해 이동. 충전 시 역반응.
전극 소재 구조 파괴 없이 정반응·역반응 반복 → 장수명의 핵심.

전지의 분류 체계
물리전지 vs 화학전지 / 일차전지 vs 이차전지
전지 분류 전체 트리
🌞

물리전지

물리 반응으로 전기를 생성.
태양전지: 빛 → 전기
열전지: 열 → 전기
원자력전지: 핵분열 → 전기

비화학적
⚗️

화학전지

화학 반응으로 전기를 생성.
일차전지: 충전 불가 (비가역)
이차전지: 충전 가능 (가역)
연료전지: 연속 공급 연료

산화·환원 반응
🔌

건전지 vs 습전지

건전지(Dry Cell): 전해액에 젤라틴·전분 첨가, 페이스트화.
습전지(Wet Cell): 전해액이 유동 상태로 과잉 포함.

구조 차이
♻️

일차 vs 이차전지

일차전지: 비가역 반응, 한 번만 방전
예) 알칼리, 망간, 수은, 리튬 일차전지
이차전지: 가역 반응, 반복 충·방전
예) 납축전지, NiCd, Ni-MH, LIB

대표 일차전지
아연-탄소 전지 (건전지) — 대표적 1차전지, 약 1.5V ▼
양극: 탄소막대(집전체) + 이산화망간(MnO₂)
음극: 아연(Zn)

산화 전극: Zn(s) → Zn²⁺(aq) + 2e⁻
환원 전극: 2MnO₂ + 2NH₄⁺ + 2e⁻ → Mn₂O₃ + 2NH₃ + H₂O

특징: 매우 저렴, 모양·크기 다양. 시간이 지남에 따라 전압 감소하는 단점.
알칼리 전지 — 약 1.5V, 장수명 ▼
산화 전극(음극): Zn(s) + 2OH⁻ → ZnO(s) + H₂O + 2e⁻
환원 전극(양극): MnO₂ + H₂O + 2e⁻ → Mn₂O₃ + 2OH⁻
전체 반응 ΔG = −275.9 kJ/mol, E = +1.43 V

방전 과정은 자발적 반응(ΔG < 0), 갈바닉 셀 원리로 작동.
최초의 전지 — 볼타전지 (1800)

볼타전지 (Voltaic Cell / Galvanic Cell)

음극 · 산화
Zn(s) → Zn²⁺(aq) + 2e⁻
양극 · 환원
2H⁺(aq) + 2e⁻ → H₂(g)

양극: 구리(Cu) / 음극: 아연(Zn) / 전해질: 묽은 황산(H₂SO₄)
금속이 이온과 전자로 분리될 때 고유 전압으로 전류 흐름. 볼트(V) 단위가 볼타 이름에서 유래.

산화·환원 반응 (Redox Reaction)
이차전지의 핵심 화학 반응 원리
기본 정의
🔴

산화 (Oxidation)

전자를 잃는 반응.
산화수 증가.
산화제는 상대를 산화시키고 자신은 환원됨.

Zn → Zn²⁺ + 2e⁻
전자 방출
🔵

환원 (Reduction)

전자를 얻는 반응.
산화수 감소.
환원제는 상대를 환원시키고 자신은 산화됨.

Cu²⁺ + 2e⁻ → Cu
전자 획득
🔢

산화수 (Oxidation Number)

전자가 완전히 이전될 때 원자가 가질 것으로 예상되는 전하.
산화수 변화 확인 = 산화-환원 반응 확인.

감소 = 환원 / 증가 = 산화
⚖️

산화-환원 동시 발생

산화와 환원은 항상 동시에 일어남.
한쪽이 전자를 잃으면, 반드시 다른 쪽이 그 전자를 받음.

잃은 전자 = 얻은 전자
전기화학 전지의 두 종류
🔋

갈바닉 셀 (Galvanic Cell)

자발적 전기화학 반응을 통해 전기에너지를 발생시키는 장치.
ΔG < 0 → E > 0
모든 일반 전지(방전)가 이 원리.

자발 반응 · 방전
🔌

전해 셀 (Electrolytic Cell)

외부 전기에너지를 공급받아 비자발적 반응을 일으키는 장치.
ΔG > 0 → 외부 에너지 필요
이차전지 충전, 전기도금에 활용.

비자발 반응 · 충전
표준 전극 전위 & 기전력(EMF)

표준 전지 전위 계산

환원 전극 (양극)
Cu²⁺ + 2e⁻ → Cu    E° = +0.34 V
산화 전극 (음극)
Zn → Zn²⁺ + 2e⁻    E° = −(−0.76) V

E°전지 = E°(환원전극) − E°(산화전극) = 0.34 − (−0.76) = 1.10 V

표준 수소 전극(SHE): 2H⁺ + 2e⁻ → H₂ 를 0 V 기준으로 설정.
E°전지 > 0 → 자발적 반응 (볼타전지), E°전지 < 0 → 비자발 (전해전지)

LIB에서 산화·환원 정리
상태음극 (흑연)양극 (LCO)전자 흐름
방전 LiₓC → C + xLi⁺ + xe⁻ (산화) Li₁₋ₓCoO₂ + xLi⁺ + xe⁻ → LiCoO₂ (환원) 음극 → 외부 → 양극
충전 C + xLi⁺ + xe⁻ → LiₓC (환원) LiCoO₂ → Li₁₋ₓCoO₂ + xLi⁺ + xe⁻ (산화) 양극 → 외부 → 음극

열 평형의 법칙

서로 다른 온도의 물체를 접촉하면 충분한 시간 후 온도가 동일해짐.

1

에너지 보존

에너지는 생성·소멸되지 않고 형태만 변환됨.
ΔU = Q − W

2

엔트로피 증가

고립계에서 자발적 변화가 일어나면 엔트로피(무질서도)는 항상 증가.

3

절대 영도

절대 0 K에서 계의 엔트로피는 0이 됨. ΔS → 0 as T → 0 K.

깁스 자유 에너지 (Gibbs Free Energy)
ΔG < 0
자발적 반응
(방전, 갈바닉셀)
ΔG = 0
평형 상태
(E = 0)
ΔG > 0
비자발적 반응
(충전, 전해셀)

깁스 자유 에너지와 전기화학 에너지의 관계

ΔG = ΔH − TΔS
ΔG = −E · F · n
E° = −ΔG / (nF)

F = 패러데이 상수 = 96,485 C/mol (전자 1 mol의 전하량)
예) H₂ + ½O₂ → H₂O : ΔG = −237 kJ/mol → E° = 1.23 V
깁스 자유 에너지 변화량을 전압으로 변환 가능.

이차전지 역사 연대기
1780년부터 현재까지 — 전지 기술의 흐름
1780
🐸 갈바니의 개구리 실험
이탈리아 볼로냐대 루이지 갈바니(Luigi Galvani) 교수가 개구리 해부 실험에서 금속 접촉 시 다리가 경련하는 현상 관찰. "동물전기" 개념 제시. 이후 볼타가 이에 의문을 품고 연구함.
1800
⚡ 볼타전지 발명 — 최초의 전지
이탈리아 알렉산드로 볼타(Alessandro Volta)가 구리-아연 일차전지 발명. 구리 원반 + 산 용액을 적신 헝겊 + 아연 원반을 교대로 쌓아 제조. 현재 전압 단위 볼트(V)가 그의 이름에서 유래.
일차전지 전압 단위 V
1859
🔋 납축전지 — 최초의 이차전지
프랑스 물리학자 가스통 플랑테(Gaston Planté)가 발명. 납(Pb)과 황산(H₂SO₄)의 전기화학 반응 이용. 최초의 충전 가능 전지. 셀당 전압 약 2V. 현재도 자동차용 배터리로 사용 (12V = 2V × 6셀).
이차전지 최초
1899
🇸🇪 니켈카드뮴전지 (NiCd) 발명
스웨덴의 발데마르 융너(Waldemar Jungner)가 발명. 양극: NiOOH, 음극: Cd, 전해질: KOH. 1906년 스웨덴에 공장 설립. 1980년대부터 대중화. 에너지 밀도 40~60 Wh/kg. 카드뮴 독성 문제 있음.
휴대용 이차전지 시작
1967
🧪 니켈수소전지 (Ni-MH) 개발 시작
바텔제네바연구센터에서 개발 시작. 수소저장 합금을 음극으로 사용. NiCd 대비 에너지 밀도 약 2배. 저온 특성 우수, 급속 충전 가능. 세계 최초 하이브리드차 도요타 프리우스에 채택. 1989년 상용화.
에너지밀도 향상
1972
🛢️ 휘팅엄, 엑손(Exxon) 입사
유가 위기로 석유회사 엑손이 대체에너지 연구 투자. 스탠리 휘팅엄(Stanley Whittingham)이 합류. TiS₂(이황화티타늄)의 층상구조에 Li⁺을 저장하는 원리 발견 → 최초 기능적 리튬 배터리 개발 (구동전압 2V 미만). 덴드라이트 문제로 개발 중단.
1단계: 휘팅엄
1980
⚗️ 구디너프, LiCoO₂ 양극 개발
영국 옥스퍼드대 존 구디너프(John Goodenough) 교수가 금속 황화물 대신 금속 산화물을 양극 소재로 사용. 리튬코발트산화물(LiCoO₂)로 4V급 고전압 양극 개발 발표. 에너지 밀도를 두 배로 향상. "충전된 상태가 아니라 나중에 충전할 수 있다"는 개념 확립.
2단계: 구디너프 4V 고전압
1985
🏭 요시노, 최초 상용 가능 LIB 개발
일본 요시노 아키라(Yoshino Akira) 박사가 구디너프의 LiCoO₂ 양극 + 석유 코크(후에 흑연) 음극 조합으로 안정적·경량·고용량·4V 구동 이차전지 개발. 순수 리튬 금속 없이 리튬 이온만 사용 → "리튬이차전지"에서 "리튬이온전지"로 명칭 변경.
3단계: 요시노
1986
🔨 안전성 테스트 통과 — LIB의 탄생 순간
요시노 박사가 큰 철 조각을 배터리 위에 떨어뜨리는 안전 테스트에서 아무 이상 없음을 확인. 요시노 본인이 "이 순간이 리튬이온전지가 탄생한 순간"이라고 진술.
1989
🚗 Ni-MH 전지 상용화
니켈수소전지 최초 상용화. 이후 초창기 전기자동차 및 하이브리드차에 적용. 세계 최초 하이브리드차 도요타 프리우스(1997)에 탑재.
1991
📱 리튬이온전지 최초 상용화
LiCoO₂ 양극 + 흑연 음극 구성의 리튬이온전지가 처음으로 상용화(소니). 충·방전 시 리튬이온의 삽입/탈리(인터칼레이션) 반응. 흔들의자(Rocking Chair)형 전지라고도 불림. 이후 모바일 기기 혁명의 핵심 동력.
상용화 무선 혁명 시작
2000년대~
🚘 전기자동차 시대 개막
LIB 기술이 모바일 전자기기(휴대폰·태블릿·노트북)에서 전기자전거, 전동킥보드, EV로 확장. 테슬라 배터리팩(68 kWh) = 18650 셀 약 6,300개. 에너지 밀도 목표: Egrav > 400 Wh/kg.
2019
🏆 노벨화학상 — 리튬이온전지
휘팅엄(Whittingham) · 구디너프(Goodenough) · 요시노 아키라(Yoshino Akira) 공동 수상. "화석 연료 없는 사회를 위한 필수 기술"로 평가. 리튬이온전지 개발에 기여한 세 과학자의 업적 인정.

납축전지 (Lead-Acid Battery)

음극판 · 산화 (방전)
Pb + H₂SO₄ → PbSO₄ + 2H⁺ + 2e⁻
양극판 · 환원 (방전)
PbO₂ + H₂SO₄ + 2H⁺ + 2e⁻ → PbSO₄ + 2H₂O

충전 시: 역반응 진행 (전해셀 원리). 자동차용 12V = 2V × 6셀 직렬 연결.

니켈카드뮴(NiCd) 반응식 — 방전

음극 · 산화
Cd + 2OH⁻ → Cd(OH)₂ + 2e⁻
양극 · 환원
2NiO(OH) + 2H₂O + 2e⁻ → 2Ni(OH)₂ + 2OH⁻
